# BLACK JACK 292

Custom Blackjack variant with 7 starting hands, hidden/revealed cards,
and a full set of special actions (hit, stand, double down, split,
insurance, surrender).

## Setup

1. `npm install`
2. Copy `.env.example` to `.env` and fill in your `DATABASE_URL`
3. `node server/db/init.js` (creates the database tables)
4. `npm start`
